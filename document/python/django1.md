# 설치

- 윈도우 powershell (Python 사전 설치 필수)
```
pip install django
```

- Django 프로젝트 생성
```
django-admin startproject <프로젝트이름>
```

- Django 서버 실행
```
(프로젝트폴더에서) python manage.py runserver
```

- Django 서버 접속
```
(웹브라우저에서) 17.0.0.1:8000 or loaclhost:8000
```

- app 만들기
```
(프로젝트폴더에서) python manage.py startapp <앱이름>
```
